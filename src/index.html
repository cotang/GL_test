Часть 2: форма обратной связи

При нажатии на кнопку "Обратная связь" необходимо открывать всплывающее окно с формой (Ajax), содержащей следующие поля:

  1. ФИО (обязательное поле)
  2. Телефон (обязательное поле, если не заполнено поле "Email")
  3. Email (обязательное поле, если не заполнено поле "Телефон")
  4. Текст сообщения (обязательное поле)
  5. Отправить

Форма должна валидироваться на клиенте, после отправки формы, с сервера приходит ответ в формате JSON.

В случае успеха:

{
    "success": true,
    "msg": "Сообщение отправлено, в ближайшее время наш менеджер свяжется с Вами."
}

В случае неудачи:

{
    "success": false,
    "fields": [
        {
            "name": "email"
            "error": "Введенный Вами e-mail не найден"
        }
    ]
}

или:

{
    "success": false,
    "msg": "Ошибка отправки формы"
}
